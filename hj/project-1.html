ImagePix – 이미지 검색 서비스 프론트엔드 개발

기간: 2025.04.01 ~ 2025.05.31

HTML CSS JavaScript AJAX PHP MySQL AWS EC2

요약

- 유저가 키워드를 검색하여 원하는 이미지를 쉽게 찾을 수 있는 이미지 추적 플랫폼 개발
- 기존 실사 이미지뿐만 아니라 AI가 만든 독창적인 이미지도 함께 제공
- 리눅스 기반 웹 서버 및 DB 서버를 구축하고, 아파치 및 MySQL을 연동하여 독립적인 서비스 환경 구성

역할

- 검색 및 정렬 기능 구현 (추천순, 인기순)
- 유동적 상단 UI 및 이미지 오버레이 효과 적용
- 이미지 상세 보기 및 구매 기능 개발
- Apache, MySQL 기반 서버 환경 직접 구성 및 설정
- 웹 서비스 배포 및 운영 환경 초기 세팅 참여

성과

- 사용자 친화적인 UI/UX 제공으로 편리한 이미지 검색 환경 구축
- AI 이미지와 실사 이미지를 함께 제공하여 차별화된 콘텐츠 확보
- 자체 서버 환경 구축 및 관리 경험을 통해 운영 안정성과 확장성 고려한 시스템 설계 역량 강화
