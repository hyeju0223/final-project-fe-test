ESGX – ESG 추천 플랫폼 프론트엔드 개발

기간: 2025.04.01 ~ 2025.05.31

HTML CSS JavaScript AJAX PHP MySQL AWS EC2

요약

기업 맞춤형 다양한 ESG 활동을 추천하고 아카이빙하는 플랫폼으로서, AI 기반 스마트 매칭 기능과 뉴스 자동 요약 및 배포 기능을 포함

역할

- 카테고리 및 필터별 ESG 조회 기능 개발
- ESG 카드 상세 화면 및 게시물 내비게이션 구현
- EC2 서버 환경에 배포 및 운영

성과

- ESG 정보 접근성 향상
- 사용자 경험을 고려한 UI/UX 개선
