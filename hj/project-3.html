청운속기사무소 - 프론트엔드 개발

기간: 2025.04.01 ~ 2025.05.31

HTML CSS JavaScript jQuery PHP Apache GCS AWS EC2

요약

사용자가 음성파일을 올리고 의뢰를 하면, 대용량 음성 파일이 구글 클라우드(GCS)에 업로드되어 서버 유지비용이 매우 저렴한 운영방식으로 설계된 녹취록 의뢰 사이트 개발

역할

- 서비스 소개 페이지 UI 개발
- GCS(Google Cloud Storage)를 활용한 대용량 음성파일 업로드 기능 구현
- EC2 서버 환경에 배포 및 운영

성과

- 효율적인 대용량 음성파일 처리 및 업로드 시스템 설계로 서버 비용 절감에 기여함.
- 사용자 친화적인 UI/UX 설계로 서비스 편의성 향상
