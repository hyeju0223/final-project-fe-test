우수렌트카 – 렌트 예약 웹 개발

기간: 2025.04.01 ~ 2025.05.31

HTML CSS JavaScript AJAX PHP MySQL AWS EC2

요약

- 커스텀 달력 기반으로 대차일/반납일에 따라 자동 할인율 적용이 가능한 실시간 차량 예약 시스템
- 사용자 편의를 고려한 예약, 검색, 결제, 정보 확인 등 통합 UI 개발

- EC2 서버 환경에 배포 및 운영

성과

- 다양한 조건별 차량 탐색 및 정보 접근성 향상
- 고객 후기 슬라이더 및 토글 기능으로 UX 향상
